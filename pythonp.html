PYTHON Programming notes

Python is a high-level, interpreted programming language known for its readability and simplicity. Created by Guido van Rossum and first released in 1991, Python emphasizes code clarity and allows programmers to express concepts in fewer lines of code compared to other languages.

Key Features: Readable Syntax: Python's syntax is designed to be intuitive and easy to read, making it accessible for beginners.

Versatility: Python supports multiple programming paradigms, including procedural, object-oriented, and functional programming.

Extensive Libraries: It has a rich ecosystem of libraries and frameworks, such as NumPy for numerical computations, Pandas for data analysis, and Django for web development, enabling developers to work on a wide range of applications.

Cross-Platform: Python can run on various operating systems, including Windows, macOS, and Linux, without requiring modification to the code.

Strong Community Support: Python has a large, active community, which contributes to its continuous improvement and provides extensive documentation and resources.

Control flow statements are used to dictate the order in which statements are executed in a program. Python primarily uses conditional statements and loops for controlling the flow of execution.

1. Conditional Statements

Conditional statements allow you to execute certain blocks of code based on specific conditions.

  • if Statement: Executes a block of code if the condition is true.
    • if condition:
            # code to execute if condition is true
  • else Statement: Executes a block of code if the preceding if condition is false.
    • if condition:
            # code to execute if condition is true
        else:
            # code to execute if condition is false
  • elif Statement: Short for "else if," it allows checking multiple conditions.
    • if condition1:
            # code for condition1
        elif condition2:
            # code for condition2
        else:
            # code if none of the above conditions are true
  • Example: 
    age = 20
        if age < 18:
            print("Minor")
        elif age < 65:
            print("Adult")
        else:
            print("Senior")

2. Looping Statements

Looping statements allow you to repeat a block of code multiple times.

  • for Loop: Iterates over a sequence (like a list or range).
    • for item in sequence:
            # code to execute for each item
  • Example: 
    fruits = ["apple", "banana", "cherry"]
        for fruit in fruits:
            print(fruit)
  • while Loop: Repeats a block of code as long as a specified condition is true.
    • while condition:
            # code to execute while condition is true
  • Example: 
    count = 0
        while count < 5:
            print(count)
            count += 1

3. Control Statements within Loops

  • break Statement: Exits the loop prematurely.
    • for i in range(10):
            if i == 5:
                break  # Exits the loop when i is 5
            print(i)
  • continue Statement: Skips the current iteration and continues with the next iteration.
    • for i in range(10):
            if i % 2 == 0:
                continue  # Skips even numbers
            print(i)  # Prints odd numbers
  • pass Statement: Does nothing; it is a placeholder.
    • for i in range(5):
            if i == 2:
                pass  # Placeholder for future code
            print(i)


Functions are reusable blocks of code that perform a specific task. They help in organizing code, making it modular and easier to maintain.

1. Defining a Function

Syntax:

def function_name(parameters):
            # code to execute
            return value  # optional

Example:

def greet(name):
            print("Hello, " + name + "!")

2. Calling a Function

To execute a function, simply call it by its name and provide any required arguments.

greet("Alice")  # Output: Hello, Alice!

3. Function Parameters

Positional Parameters: Parameters are assigned values based on their position.

def add(a, b):
            return a + b

        result = add(5, 3)  # Output: 8

Default Parameters: Parameters can have default values.

def greet(name="Guest"):
            print("Hello, " + name + "!")

        greet()        # Output: Hello, Guest!
        greet("Alice")  # Output: Hello, Alice!

Keyword Parameters: You can pass parameters by explicitly naming them.

def display_info(name, age):
            print(f"Name: {name}, Age: {age}")

        display_info(age=30, name="Bob")  # Output: Name: Bob, Age: 30

4. Return Statement

A function can return a value using the return statement. If no return statement is provided, the function returns None by default.

def multiply(x, y):
            return x * y

        result = multiply(4, 5)  # Output: 20

5. Function Scope

Local Variables: Variables defined inside a function are local to that function.

def my_function():
            local_var = 10
            print(local_var)

        my_function()  # Output: 10
        # print(local_var)  # Error: NameError

Global Variables: Variables defined outside any function are global and can be accessed inside functions unless shadowed by a local variable.

global_var = 5

        def my_function():
            print(global_var)  # Accessing global variable

        my_function()  # Output: 5

6. Anonymous Functions (Lambda Functions)

A lambda function is a small, unnamed function defined using the lambda keyword. It can take any number of arguments but can only have one expression.

square = lambda x: x ** 2
        print(square(4))  # Output: 16

7. Higher-Order Functions

Functions that take other functions as arguments or return functions as their result.

def apply_function(func, value):
            return func(value)

        result = apply_function(square, 5)  # Output: 25


Data structures are essential for organizing and managing data efficiently. Python provides several built-in data structures that enable various ways of storing and manipulating data.

1. Lists

Description: An ordered, mutable collection of items, allowing duplicate elements.

my_list = [1, 2, 3, 4, 5]

Common Operations:

  • Accessing elements: my_list[0] (Output: 1)
  • Adding elements: my_list.append(6)
  • Removing elements: my_list.remove(3)
  • Slicing: my_list[1:4] (Output: [2, 3, 4])

2. Tuples

Description: An ordered, immutable collection of items, allowing duplicate elements.

my_tuple = (1, 2, 3, 4, 5)

Common Operations:

  • Accessing elements: my_tuple[0] (Output: 1)
  • Slicing: my_tuple[1:4] (Output: (2, 3, 4))

3. Sets

Description: An unordered collection of unique items; duplicates are not allowed.

my_set = {1, 2, 3, 4, 5}

Common Operations:

  • Adding elements: my_set.add(6)
  • Removing elements: my_set.remove(3)
  • Set operations:
    • Union: set1 | set2
    • Intersection: set1 & set2
    • Difference: set1 - set2

4. Dictionaries

Description: An unordered collection of key-value pairs, where keys must be unique.

my_dict = {'name': 'Alice', 'age': 25}

Common Operations:

  • Accessing values: my_dict['name'] (Output: 'Alice')
  • Adding key-value pairs: my_dict['city'] = 'New York'
  • Removing key-value pairs: del my_dict['age']
  • Keys and values: my_dict.keys(), my_dict.values()

5. Strings

Description: Immutable sequences of characters. While not a data structure like the others, they are fundamental to data manipulation.

my_string = "Hello, World!"

Common Operations:

  • Accessing characters: my_string[0] (Output: 'H')
  • Slicing: my_string[0:5] (Output: 'Hello')
  • String methods: .upper(), .lower(), .replace(), .split()


File handling is an essential aspect of programming that allows you to read from and write to files. Python provides built-in functions and methods to facilitate file operations.

1. Opening a File

To work with a file, you first need to open it using the open() function.

file_object = open('filename', 'mode')

Modes:

  • 'r': Read (default) - Opens a file for reading.
  • 'w': Write - Opens a file for writing (overwrites if exists).
  • 'a': Append - Opens a file for appending (writes at the end).
  • 'b': Binary - Opens a file in binary mode.
  • 'x': Exclusive creation - Fails if the file already exists.

Example:

file = open('example.txt', 'r')

2. Reading from a File

You can read data from a file using various methods:

  • read(size): Reads the entire file or a specified number of bytes. 
    content = file.read()  # Reads the entire file
  • readline(): Reads a single line from the file. 
    line = file.readline()
  • readlines(): Reads all lines and returns them as a list. 
    lines = file.readlines()

3. Writing to a File

To write data to a file, you can use the following methods:

  • write(string): Writes a string to the file. 
    file = open('example.txt', 'w')
    file.write("Hello, World!")
  • writelines(list): Writes a list of strings to the file. 
    lines = ["Line 1\n", "Line 2\n"]
    file.writelines(lines)

4. Closing a File

It is crucial to close a file after completing operations to free up system resources.

file.close()

5. Using with Statement

Using the with statement is a best practice for file handling as it automatically closes the file after the block of code is executed.

with open('example.txt', 'r') as file:
            content = file.read()
            # No need to explicitly close the file

6. Exception Handling

File operations can lead to exceptions (e.g., file not found). Use try and except blocks to handle potential errors.

try:
                with open('nonexistent.txt', 'r') as file:
                    content = file.read()
            except FileNotFoundError:
                print("File not found.")


Error handling is crucial in programming as it allows developers to gracefully manage exceptions and errors that may occur during the execution of a program. In Python, this is primarily done using try, except, finally, and else blocks.

1. Exceptions

Definition: Exceptions are events that disrupt the normal flow of a program. Common exceptions include ZeroDivisionError, TypeError, ValueError, and FileNotFoundError.

Raising Exceptions: You can raise exceptions using the raise statement.

raise ValueError("Invalid value")

2. Using try and except

The try block lets you test a block of code for errors, while the except block allows you to handle the error.

Syntax:

try:
            # Code that may cause an exception
        except SomeException:
            # Code to handle the exception

Example:

try:
            result = 10 / 0
        except ZeroDivisionError:
            print("You can't divide by zero!")

3. Catching Multiple Exceptions

You can handle multiple exceptions in a single except block by specifying a tuple of exceptions.

Example:

try:
            x = int(input("Enter a number: "))
            result = 10 / x
        except (ValueError, ZeroDivisionError) as e:
            print(f"An error occurred: {e}")

4. Using else Block

The else block can be used to execute code that should run only if the try block did not raise an exception.

Example:

try:
            result = 10 / 2
        except ZeroDivisionError:
            print("Division by zero!")
        else:
            print(f"Result is {result}")

5. Using finally Block

The finally block will execute regardless of whether an exception was raised or not. Itâ€™s commonly used for cleanup actions.

Example:

try:
            file = open('example.txt', 'r')
            content = file.read()
        except FileNotFoundError:
            print("File not found.")
        finally:
            file.close()  # This will execute no matter what

6. Raising Exceptions

You can raise exceptions manually using the raise statement. This is useful for enforcing conditions in your code.

Example:

def divide(a, b):
            if b == 0:
                raise ValueError("The denominator cannot be zero.")
            return a / b

        try:
            divide(10, 0)
        except ValueError as e:
            print(e)


Object-oriented programming is a programming paradigm that uses "objects" to represent data and methods to manipulate that data. OOP helps organize code in a way that makes it easier to manage and reuse.

1. Key Concepts of OOP

  • Classes: A class is a blueprint for creating objects. It defines attributes (data) and methods (functions) that the created objects will have.
  • Objects: An instance of a class. Objects can contain data (attributes) and can perform actions (methods).
  • Encapsulation: The bundling of data (attributes) and methods that operate on that data within a single unit (class). It restricts direct access to some of the object's components.
  • Inheritance: A mechanism that allows one class to inherit attributes and methods from another class, promoting code reusability.
  • Polymorphism: The ability to present the same interface for different underlying data types. In Python, this is achieved through method overriding and duck typing.

2. Defining a Class

You can define a class in Python using the class keyword.

Syntax:

class ClassName:
            # Class attributes and methods

Example:

class Dog:
            def __init__(self, name, age):
                self.name = name  # Attribute
                self.age = age    # Attribute

            def bark(self):      # Method
                print("Woof!")

3. Creating an Object

To create an object from a class, you call the class as if it were a function.

Example:

my_dog = Dog("Buddy", 3)  # Creating an object of the Dog class

4. Accessing Attributes and Methods

You can access an object's attributes and methods using the dot (.) notation.

Example:

print(my_dog.name)  # Accessing an attribute
        my_dog.bark()       # Calling a method

5. Inheritance

You can create a new class that inherits attributes and methods from an existing class.

Syntax:

class ChildClass(ParentClass):
            # Additional attributes and methods

Example:

class Puppy(Dog):  # Puppy inherits from Dog
            def wag_tail(self):
                print("Wagging tail!")
            
        my_puppy = Puppy("Max", 1)
        my_puppy.bark()   # Inherited method
        my_puppy.wag_tail()  # Own method

6. Polymorphism

Polymorphism allows methods to do different things based on the object calling them. This is often achieved through method overriding.

Example:

class Cat:
            def bark(self):
                print("Meow!")

        def make_sound(animal):
            animal.bark()  # Different animals can use the same method name

        my_cat = Cat()
        make_sound(my_cat)  # Outputs: Meow!

7. Encapsulation

Encapsulation is achieved by using private attributes and methods. You can prefix an attribute or method with an underscore (_) or double underscore (__) to make it private.

Example:

class BankAccount:
            def __init__(self, balance):
                self.__balance = balance  # Private attribute

            def deposit(self, amount):
                self.__balance += amount

            def get_balance(self):
                return self.__balance


A module is a file containing Python code that can define functions, classes, and variables. Modules help organize code into manageable sections, promoting code reuse and maintaining a clean namespace.

1. What is a Module?

Definition: A module is a file with a .py extension that contains Python definitions and statements.

Purpose: Modules allow you to logically organize your Python code and reuse code across different programs.

2. Creating a Module

You can create a module simply by writing Python code in a file and saving it with a .py extension.

Example: Create a file named mymodule.py with the following content:

def greet(name):
            return f"Hello, {name}!"

        PI = 3.14159

3. Importing a Module

You can use the import statement to include a module in your program.

Syntax:

import module_name

Example:

import mymodule

        print(mymodule.greet("Alice"))  # Outputs: Hello, Alice!
        print(mymodule.PI)               # Outputs: 3.14159

4. Importing Specific Functions or Variables

You can import specific functions or variables from a module to avoid namespace clutter.

Syntax:

from module_name import function_name

Example:

from mymodule import greet

        print(greet("Bob"))  # Outputs: Hello, Bob!

5. Importing All Functions from a Module

You can import everything from a module using the asterisk (*), but this is generally discouraged as it can lead to unclear code.

Example:

from mymodule import *

        print(greet("Charlie"))  # Outputs: Hello, Charlie!

6. Aliasing Modules

You can give a module or a function an alias using the as keyword, which can simplify references to it.

Example:

import mymodule as mm

        print(mm.greet("David"))  # Outputs: Hello, David!

7. Standard Modules

Python comes with a set of built-in modules (also called standard libraries) that provide various functionalities, such as:

  • math: Mathematical functions (e.g., math.sqrt()).
  • random: Generate random numbers (e.g., random.randint()).
  • os: Interact with the operating system (e.g., os.getcwd()).
  • sys: Access system-specific parameters and functions (e.g., sys.exit()).

8. Creating Packages

A package is a collection of modules organized in a directory hierarchy. To create a package, follow these steps:

  1. Create a directory for your package.
  2. Add an __init__.py file (can be empty) to the directory to mark it as a package.
  3. Add your module files inside this directory.

Example Directory Structure:

mypackage/
            __init__.py
            module1.py
            module2.py

9. Importing Packages

You can import modules from a package using the dot notation.

Example:

from mypackage import module1
