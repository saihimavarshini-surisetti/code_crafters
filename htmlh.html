HTML notes

HTML, or HyperText Markup Language, is the standard markup language used to create and design web pages and web applications. It structures content on the web by using a series of elements or tags. Here are some key points about HTML:

Structure: HTML defines the structure of a web page using elements like headings, paragraphs, links, images, lists, and more.

Tags: HTML is composed of tags that denote different types of content. For example:

< h1 > to < h6 > for headings

< p >

for paragraphs

< a > for hyperlinks

< img > for images

Attributes: Tags can have attributes that provide additional information. For example, the tag can include an href attribute to specify the URL of the link.

Hierarchy: HTML documents are structured hierarchically, with nested elements. The basic structure of an HTML document includes:

< !DOCTYPE html >: Declaration to specify the document type.

< html >: Root element that wraps the entire content.

< head >: Contains meta-information, links to stylesheets, and scripts.

< body >: Contains the content displayed on the web page.

Semantics: HTML5 introduced semantic elements that clearly describe their meaning in a human- and machine-readable way, like < article >, < section >, < header >, and < footer>.

Rendering: Browsers interpret HTML to display web pages visually, allowing users to interact with the content.



HTML (HyperText Markup Language): The standard language for creating web pages.

Purpose: To structure web content using tags and elements.

Basic Structure of an HTML Document

Every HTML file has a basic structure:

            <!DOCTYPE html> <!-- Defines the document type -->
            <html> <!-- Root element of the document -->
            <head> <!-- Contains meta-information like title, styles, etc. -->
                <title>Page Title</title> <!-- Title of the web page shown in browser tab -->
            </head>
            <body> <!-- The content of the page that is visible in the browser -->
                <h1>This is a Heading</h1> <!-- Heading element -->
                <p>This is a paragraph.</p> <!-- Paragraph element -->
            </body>
            </html>


HTML Elements

An HTML element consists of an opening tag, content (if applicable), and a closing tag. It defines the structure and content of a webpage.

  • Opening Tag: Marks the start of the element (e.g., <p>).
  • Content: Text or nested HTML elements inside the opening and closing tags.
  • Closing Tag: Marks the end of the element (e.g., </p>).

Example:

            <p>This is a paragraph.</p>

HTML Tags

HTML tags are the building blocks of elements. Tags are written inside angle brackets (< >) and often come in pairs: an opening tag and a closing tag.

Types of Tags:

  • Paired Tags: Most elements use paired tags with an opening and closing tag.
    • <h1>Main Heading</h1>
  • Self-closing Tags: Tags that do not need a closing tag (e.g., <img>).
    • <img src=image.jpg alt=Image>

Common HTML Elements and Tags

  • Headings: <h1> to <h6>
    • <h1>Main Heading</h1>
  • Paragraphs: <p>
    • <p>This is a paragraph.</p>
  • Links: <a>
    • <a href=https://example.com>Visit</a>
  • Images: <img>
    • <img src=logo.png alt=Logo>
  • Lists:
    • Unordered List: <ul>
    • Ordered List: <ol>
    <ul><li>Item 1</li></ul>

Attributes

Attributes provide additional information about elements, written inside the opening tag.

<a href=https://example.com>Link</a>

href: Specifies the link destination.

Self-closing Elements

These elements don't need closing tags.

  • <img>
  • <br>
  • <hr>


What are HTML Attributes?

HTML attributes provide additional information about HTML elements. They modify behavior, define properties, and enhance functionality.

Key Points:

  • Attributes are always included in the opening tag.
  • They follow the syntax: name="value".
  • Most HTML elements can have attributes.

Common HTML Attributes

  • href (for links): Specifies the URL of the linked document.
    • <a href=https://www.example.com\>Visit Example</a>
  • src (for images): Specifies the path to the image file.
    • <img src=image.jpg alt=Description of the image>
  • alt (for images): Provides alternative text for an image.
    • <img src=logo.png alt=Company Logo>
  • id: Specifies a unique identifier for an element.
    • <h1 id=main-heading>Welcome</h1>
  • class: Specifies one or more class names for an element.
    • <p class=intro-text>This is a paragraph.</p>
  • style: Specifies inline CSS styles for an element.
    • <p style=color: blue; font-size: 20px;>This is styled text.</p>
  • title: Adds a tooltip when hovering over an element.
    • <a href=https://example.com title=Visit Example Website>Example</a>
  • target (for links): Specifies where to open the linked document.
    • <a href=https://example.com target=_blank>Open in new tab</a>

Global Attributes

These attributes can be applied to any HTML element:

  • id: Unique identifier for an element.
  • class: Specifies one or more class names.
  • style: Adds inline styles.
  • title: Adds a tooltip on hover.
  • data-*: Custom data attributes for storing extra data.
<div data-info=custom-data>Content</div>

Boolean Attributes

Boolean attributes do not need a value; their presence is enough to set them to true.

  • disabled (for form elements):
    • <input type=text disabled>
  • checked (for checkboxes and radio buttons):
    • <input type=checkbox checked>


What are HTML Forms?

An HTML form is used to collect user input and submit it to a server for processing. Forms are essential for creating interactive features like search bars, login pages, or feedback forms.

Key Elements of a Form:

  • <form>: The container for form elements.
  • <input>: A versatile element used to capture user data (e.g., text, passwords, buttons).
  • <textarea>: A multi-line input field.
  • <select>: A dropdown list for selecting options.
  • <button>: Used to submit or reset form data.

Basic Form Structure

            <form action=/submit method=POST>
                <!-- Form fields go here -->
            </form>

action: Specifies the URL where form data is sent.

method: Specifies how data is sent (e.g., GET or POST).

Form Input Types

  • Text Input: Used for single-line text inputs.
    • <input type=text name=username>
  • Password Input: Masks the input for password fields.
    • <input type=password name=password>
  • Radio Buttons: Allows the user to select one option from a group.
    • <input type=radio name=gender value=male> Male
    <input type=radi name=gender value=female> Female
  • Checkboxes: Allows multiple selections.
    • <input type=checkbox name=subscribe value=newsletter> Subscribe to newsletter
  • Submit Button: Sends form data to the server.
    • <input type=submit value=Submit>

Textarea

For multi-line text input, use the <textarea> tag.

<textarea name=comments rows=4 cols=50;></textarea>

Select Dropdown

The <select> element allows the user to select one or more options from a dropdown list.

            <select name=country>
                <option value=usa>USA</option>
                <option value=canada>Canada</option>
            </select>

Labels

The <label> element provides a clickable label for form controls, improving accessibility.

            <label for=username>Username:</label>
            <input type=text id=username name=username>

Form Validation

HTML5 includes built-in validation attributes that help ensure data is correctly entered before submission.

  • required: Ensures the field is filled out.
    • <input type=text name=email required>
  • pattern: Validates the input against a regular expression.
    • <input type=text name=phone pattern=[0-9]{10}>
  • min and max: Define numeric range constraints.
    • <input type=number name=age min=18 max=100>

File Upload

The <input type=file> allows users to upload files.

<input type=file" name=resume>

Form Methods

  • GET: Sends form data appended to the URL (used for search forms).
    • <form action=/search method=GET></form>
  • POST: Sends form data as a separate HTTP request (used for sensitive data like login forms).
    • <form action=/login method=POST></form>


HTML Tables

What are HTML Tables?

An HTML table is used to display data in a structured format, consisting of rows and columns. Tables are useful for presenting tabular data such as schedules, comparisons, or statistics.

Key Elements of a Table:

  • <table>: The container for the entire table.
  • <tr>: Represents a table row.
  • <th>: Represents a table header cell (usually bold and centered).
  • <td>: Represents a table data cell (regular data).

Basic Table Structure

            <table>
                <tr>
                    <th>Header 1</th>
                    <th>Header 2</th>
                </tr>
                <tr>
                    <td>Data 1</td>
                    <td>Data 2</td>
                </tr>
            </table>

Attributes of Tables

  • border: Specifies the border size of the table (deprecated in HTML5, use CSS instead).
  • cellspacing: Defines the space between cells (deprecated in HTML5, use CSS instead).
  • cellpadding: Defines the space between the cell border and its content (deprecated in HTML5, use CSS instead).
  • width: Sets the width of the table (use CSS for styling).

Table Example

            <table border=1>
                <tr>
                    <th>Name</th>
                    <th>Age</th>
                    <th>City</th>
                </tr>
                <tr>
                    <td>John</td>
                    <td>25</td>
                    <td>New York</td>
                </tr>
                <tr>
                    <td>Mary</td>
                    <td>30</td>
                    <td>Los Angeles</td>
                </tr>
            </table>

Spanning Rows and Columns

  • Row Span: Use the rowspan attribute to make a cell span multiple rows.
    • <td rowspan=2>Spanning Cell</td>
  • Column Span: Use the colspan attribute to make a cell span multiple columns.
    • <td colspan=2>Spanning Cell</td>

Table Caption

The <caption> element provides a title or description for the table.

            <table>
                <caption>Employee Data</caption>
                ...
            </table>

Styling Tables

CSS can be used to style tables for better presentation.

            table {
                width: 100%;
                border-collapse: collapse;
            }

            th, td {
                border: 1px solid black;
                padding: 8px;
                text-align: left;
            }


What is Semantic HTML?

Semantic HTML refers to the use of HTML markup that conveys meaning about the content it contains. This approach improves the accessibility and searchability of web pages by helping browsers and search engines understand the structure and context of the content.

Why Use Semantic HTML?

  • Accessibility: Enhances the experience for users with disabilities by allowing screen readers to interpret the content correctly.
  • SEO Benefits: Improves search engine optimization (SEO) by helping search engines better understand the content.
  • Maintainability: Makes code easier to read and maintain by providing clear and descriptive elements.
  • Browser Compatibility: Helps browsers render the content more accurately.

Common Semantic HTML Elements

<header>

Represents introductory content, typically containing navigational links or logos.

            <header>
                <h1>Website Title</h1>
                <nav>
                    <ul>
                        <li><a href=#home>Home</a></li>
                        <li><a href=;#about>About</a></li>
                    </ul>
                </nav>
            </header>

<nav>

Represents a section of navigation links.

            <nav>
                <ul>
                    <li><a href=#services>Services</a></li>
                    <li><a href=#contact>Contact</a></li>
                </ul>
            </nav>

<main>

Represents the main content of the document, excluding headers, footers, and sidebars.

            <main>
                <h2>Main Content Heading</h2>
                <p>This is the main content of the page.</p>
            </main>

<section>

Represents a thematic grouping of content, typically with a heading.

            <section>
                <h2>About Us</h2>
                <p>Details about the organization.</p>
            </section>

<article>

Represents a self-contained piece of content, like a blog post or news article.

            <article>
                <h2>Blog Post Title</h2>
                <p>This is the content of the blog post.</p>
            </article>

<aside>

Represents content that is related to the main content but is separate, like a sidebar or a pull quote.

            <aside>
                <h3>Related Links</h3>
                <ul>
                    <li><a href=#link1>Link 1</a></li>
                    <li><a href=#link2>Link 2</a></li>
                </ul>
            </aside>

<footer>

Represents the footer of a section or page, typically containing authorship information, copyright, or links to related documents.

            <footer>
                <p>© 2024 Company Name. All rights reserved.</p>
            </footer>

Using Semantic HTML with CSS

Semantic HTML can be styled using CSS, just like traditional HTML elements. Using semantic elements helps to maintain a clear structure in your code.

Example:

            header {
                background-color: #f8f9fa;
                padding: 10px;
            }

            nav ul {
                list-style-type: none;
            }

            main {
                margin: 20px;
            }


Multimedia in HTML refers to the integration of various forms of media content, such as images, audio, video, and animations, into web pages. HTML provides several elements to embed and manage multimedia effectively.

Types of Multimedia Elements in HTML

1. Images

Used to display graphics and photos on web pages.

Element: <img>

Attributes:

  • src: Specifies the image source (URL or file path).
  • alt: Provides alternative text for the image (important for accessibility).
  • width and height: Set the dimensions of the image.
            <img src="image.jpg" alt="Description of image" width="300" height="200">

2. Audio

Used to embed sound content, such as music or voice recordings.

Element: <audio>

Attributes:

  • controls: Adds playback controls (play, pause, volume).
  • autoplay: Starts playing the audio automatically (optional).
  • loop: Repeats the audio playback (optional).
            <audio controls>
                <source src="audio.mp3" type="audio/mpeg">
                Your browser does not support the audio element.
            </audio>

3. Video

Used to embed video content.

Element: <video>

Attributes:

  • controls: Adds playback controls.
  • autoplay: Starts playing the video automatically (optional).
  • loop: Repeats the video playback (optional).
  • width and height: Set the dimensions of the video.
            <video controls width="400">
                <source src="video.mp4" type="video/mp4">
                Your browser does not support the video tag.
            </video>

4. Embedded Content

Used to embed external content such as maps or documents.

Element: <iframe>

Attributes:

  • src: Specifies the URL of the page to embed.
  • width and height: Set the dimensions of the iframe.
  • allowfullscreen: Allows the iframe to be displayed in fullscreen mode.
            <iframe src="https://www.example.com" width="600" height="400" allowfullscreen></iframe>

5. Canvas

A drawable region in HTML for graphics and animations using JavaScript.

Element: <canvas>

Attributes:

  • width and height: Set the dimensions of the canvas.
            <canvas id="myCanvas" width="200" height="100"></canvas>
